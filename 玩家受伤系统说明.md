# 玩家受伤系统使用说明

## 功能概述
实现了玩家角色受到敌人攻击时的完整受伤逻辑：
- 扣除 1 点 HP
- 2 秒无敌时间
- 无敌期间闪烁效果
- 无法控制移动
- 自动后退跳跃

## 修改的文件

### 1. PlayerHP.cs
**新增功能：**
- 无敌状态系统
- 闪烁效果
- 受伤时触发击退

**新增字段：**
- `spriteRenderer` - 角色的 SpriteRenderer（用于闪烁）
- `invincibleDuration` - 无敌时间（默认 2 秒）
- `blinkInterval` - 闪烁间隔（默认 0.1 秒）
- `isInvincible` - 是否处于无敌状态（只读属性）

**修改的方法：**
- `TakeDamage()` - 添加了无敌检测和击退触发
- 新增 `InvincibleCoroutine()` - 无敌时间和闪烁协程

### 2. PlayerMovement.cs
**新增功能：**
- 受伤击退系统
- 击退时的后退跳跃
- 击退期间禁用移动输入

**新增字段：**
- `knockbackDistance` - 击退距离（默认 2 格）
- `knockbackDuration` - 击退持续时间（默认 0.4 秒）
- `knockbackJumpHeight` - 击退跳跃高度（默认 1.5）
- `isKnockedBack` - 是否正在被击退（只读属性）

**新增方法：**
- `TriggerHitKnockback()` - 触发击退（由 PlayerHP 调用）
- `UpdateKnockback()` - 更新击退逻辑

### 3. MuscleP_AI_Movement.cs
**修改内容：**
- 添加 `isDashing` 状态标记
- 添加公共属性 `IsAttacking` 用于碰撞检测
- 在攻击冲刺开始时设置 `isDashing = true`
- 在冲刺结束时设置 `isDashing = false`

### 4. EnemyAttackCollider.cs（新文件）
**功能：**
- 检测敌人攻击时是否碰撞到玩家
- 在敌人冲刺状态下触发伤害

**字段：**
- `damageAmount` - 伤害值（默认 1）
- `playerLayer` - 玩家所在层（用于碰撞过滤）

## Unity 编辑器设置

### 玩家角色设置
1. **PlayerHP 组件：**
   - 拖入 `SpriteRenderer` 组件引用
   - 调整 `Invincible Duration`（默认 2 秒）
   - 调整 `Blink Interval`（默认 0.1 秒）

2. **PlayerMovement 组件：**
   - 调整 `Knockback Distance`（后退距离，默认 2）
   - 调整 `Knockback Duration`（击退时长，默认 0.4 秒）
   - 调整 `Knockback Jump Height`（击退跳跃高度，默认 1.5）

### 敌人角色设置
1. **添加 EnemyAttackCollider 组件：**
   - 在敌人 GameObject 上添加 `EnemyAttackCollider` 脚本
   - 设置 `Damage Amount`（伤害值，默认 1）
   - 设置 `Player Layer`（选择玩家所在层）

2. **Collider2D 设置：**
   - 确保敌人有 `Collider2D` 组件
   - 勾选 `Is Trigger` 选项
   - 调整碰撞体大小以匹配攻击范围

3. **Layer 设置：**
   - 确保玩家和敌人在不同的 Layer
   - 在 Project Settings > Physics 2D 中设置碰撞矩阵

## 工作流程

1. **敌人发起攻击：**
   - MuscleP_AI_Movement 进入攻击状态
   - 开始冲刺，`isDashing` 设为 true

2. **碰撞检测：**
   - EnemyAttackCollider 检测到与玩家碰撞
   - 检查敌人是否处于 `IsAttacking` 状态
   - 如果是，调用玩家的 `TakeDamage()`

3. **玩家受伤：**
   - PlayerHP 检查是否无敌
   - 如果不是无敌，扣除 HP
   - 开始无敌协程（闪烁 2 秒）
   - 触发 PlayerMovement 的击退

4. **击退动画：**
   - 玩家朝后方移动
   - 同时进行跳跃动画
   - 击退期间无法控制移动
   - 0.4 秒后恢复控制

5. **无敌结束：**
   - 2 秒后停止闪烁
   - 恢复正常显示
   - 可以再次受伤

## 注意事项

1. **碰撞层设置：**
   - 必须正确设置 Player Layer 和 Enemy Layer
   - 在 Physics 2D 设置中允许它们之间的碰撞

2. **Trigger 设置：**
   - 敌人的 Collider2D 必须勾选 `Is Trigger`
   - 玩家的 Collider2D 不需要是 Trigger

3. **边界限制：**
   - 击退会自动考虑世界边界
   - 不会把玩家推出地图外

4. **无敌时间：**
   - 无敌期间闪烁提示玩家
   - 无敌期间不会受到任何伤害
   - 但仍然可以主动攻击

5. **性能优化：**
   - 闪烁使用简单的 enabled 切换
   - 击退使用 FixedUpdate 保证物理稳定性

## 测试建议

1. 测试基本受伤：让敌人攻击玩家，检查 HP 是否正确扣除
2. 测试无敌时间：连续受到攻击时，只应该扣一次血
3. 测试闪烁效果：确认无敌期间玩家是否闪烁
4. 测试击退：检查后退距离和跳跃高度是否合适
5. 测试边界：在地图边缘受伤，确保不会推出边界
6. 测试移动锁定：击退期间按移动键，应该无法移动
